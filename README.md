# Gestion de Ligue de Football

Application web pour gérer une ligue de football.

## Fonctionnalités
- Ajouter des équipes
- Enregistrer des matchs
- Classement automatique

## Technologies
- Python / Flask
- MySQL
- Bootstrap

## Installation
1. Installer Flask : `pip install flask`
2. Installer mysql-connector : `pip install mysql-connector-python`
3. Créer la base de données avec `ligue.sql`
4. Lancer : `python3 app.py`
